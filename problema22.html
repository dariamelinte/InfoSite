Problema 22:

       Din fişierul text matrice.in se citesc: un număr natural n reprezentând numărul de linii, respectiv coloane ale unei matrici pătratice cu numere întregi şi cele n x n componente ale matricei (n < 100).
       Afişaţi elementele matricei sub formă tabelară în fișierul matrice.out.
       Să se verifice dacă matricea este simetrică față de diagonala secundară. În fișierul matrice.out, la linie nouă se va afișa mesajul DA în situația în care matricea este simetrică, respectiv NU în caz contrar.

Exemplu:


matrice.in:
4
1 2 3 4 5 6 7 3 8 9 6 2 0 8 5 1
matrice.out:
1 2 3 4
5 6 7 3
8 9 6 2
0 8 5 1
DA


Reozolvare:



#include < bits/stdc++.h >

using namespace std;
ifstream fin ("matrice.in");
ofstream fout ("matrice.out");

int a[101][101],n,i,j,simetric;
int main()
{

fin>>n;
for(i=1; i<=n; i++)

for(j=1; j<=n; j++)

fin>>a[i][j];

for(i=1; i<=n; i++)
{

for(j=1; j<=n; j++)

fout << a[i][j] << ' ';

fout<<'\n';

}

for(i=1; i<=n; i++)

for(j=1; j<=n; j++)
{

if(i+j==n+1)
   simetric=simetric+0;
else
if(a[i][j]==a[n-j+1][n-i+1])
   simetric++;

}


if(simetric==n*(n-1))

fout<<"DA";

else

fout<<"NU";


return 0;

}
