Prelucrarea elementelor pe linii și coloane


I. Pe linii

       Când se prelucrează elementele pentru fiecare linie în parte, se începe cu un for pentru linii, se pune acoladă, iar în interiorul acoladelor se restează ceea ce se calculează pentru fiecare linie și se parcurge linia prin forul de coloane.

Exemplu:

      Afișați suma și maximul de pe fiecare linie.

Rezolvare:


cin>>n>>m;
for(i=1; i<=n; i++)

for(j=1; j<=m; j++)

cin>>a[i][j];


for(i=1; i<=n; i++)
{

s=0; maxi=-999999;
for(j=1; j<=m; j++)
{

s=s+a[i][j];
if(a[i][j]>maxi)

maxi=a[i][j];

}
cout << s << ' ' << maxi << '\n';

}


II. Pe coloane

       Când se prelucrează elementele pentru fiecare linie în parte, se începe cu un for pentru coloane, se pune acolada, iar în interiorul acoladelor se reseteaza ceea ce se calculează pentru fiecare coloană și se parcurge coloana cu forul de linii.


Exemplu:

      Să se afișeze numărul de coloane care au toate elementele egale.

Rezolvare:


cin>>n>>m;
for(i=1; i<=n; i++)

for(j=1; j<=m; j++)

cin>>a[i][j];


for(j=1; j<=m; j++)
{

k=1;
for(i=1; i<=n; i++)

if(a[1][j]!=a[i][j])
{

k=0;
break;

}

if(k==1)

nr++;

}

cout << nr;
