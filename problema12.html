Problema 12:

       Din fişierul text tablou.in se citesc: un număr natural n reprezentând numărul de componente ale unui vector cu numere naturale nenule şi cele n componente ale vectorului (n≤100).
       Afişaţi elementele vectorului pe primul rând al fişierului text tablou.out, separate prin câte un spaţiu, în ordinea inversă citirii lor.
       Pe al doilea rând al fișierului text tablou.out afișați media aritmetică a celor n elemente citite la punctul anterior, care îndeplinesc condiția că au prima cifră egală cu ultima.

Exemplu:


tablou.in:
6
1422 131 1445 9009 171
tablou.out:
171 9009 1445 131 1422
3103.67


Rezolvare:


#include < bits/stdc++.h >

using namespace std;
ifstream fin ("tablou.in");
ofstream fout ("tablou.out");

int n,i,v[101],c1,c2;
float s,k;
int main()
{

fin>>n;
for(i=1; i<=n; i++)

fin>>v[i];

for(i=n; i>=1; i--)

fout << v[i] << ' ';

fout << endl;

for(i=1; i<=n; i++)
{

c1=v[i]%10;
c2=v[i];
while(c2>9)

c2=c2/10;

if(c1==c2)
{

s=s+v[i];
k++;

}

}
fout << s / k;
return 0;

}
