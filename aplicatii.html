Atestat - programare 2015

Problema 4:

       Din fișierul text atestat.in, se citesc trei numere naturale x, y, k; (1 < x < y < 2000000, k < 1000).
Afișați k numere prime din intervalul [x,y], pe primul rând al fișierului text atestat.out se va afișa mesajul "s-au găsit mai puține numere prime: " urmat de numărul acestora.

Exemplu:


atestat.in:
3 12 5
atestat.out:
3 5 7 11
s-au găsit mai puține numere prime: 4


Rezolvare:


#include < bits/stdc++.h >

using namespace std;
ifstream fin ("atestat.in");
ofstream fout ("atestat.out");

int x,y,nr,k,i,d,ok;
int main()
{

fin>>x>>y>>k;
for(i=x; i<=y; i++)
{

ok=0;
for(d=2; d*d<=i; d++)

if(i%d==0)
{
  ok=1;
  break;
}

if(ok==0)
{

nr++;
fout << i << ' ';

}

}

if(nr!=k)
{

fout << endl;
fout << "s-au gasit mai putine numere prime: " << nr;

}
return 0;

}
