Algoritmul de interschimbare


I. Folosim o variabilă auxiliară:


aux=x;
x=y;
y=aux;

II. Folosim adunarea și scăderea:


x=x+y;                 y=x+y;
y=x-y;       sau       x=y-x;
x=x-y;                 y=y-x;


Problemă:

      Se dau 3 numere oarecare. Să se afișeze cele 3 numere în ordine crescătoare.

Rezolvare:

#include < bits/stdc++.h >

using namespace std;
ifstream fin ("date.in");
ofstream fout ("date.out");

int a,b,c,aux;
int main()
{

fin>>a>>b>>c;
if(a>b)
{

aux=b;
b=a;
a=aux;

}
if(b>c)
{

aux=b;
b=c;
c=aux;

}
if(a>b)
{

aux=b;
b=a;
a=aux;

}
fout<< a <<' '<< b <<' '<< c;
return 0;

}
