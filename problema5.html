Problema 5:

       Două numere naturale cu proprietatea că suma divizorilor fiecărui număr (neconsiderând în acest caz numărul însuşi ca divizor) este egală cu celălalt număr plus 1, se numesc numere quasi-amiabile sau numere logodite.
       Pe fiecare linie a fişierului text numere.txt se găsesc câte 2 numere naturale. Știind că 2 numere situate pe o linie a fișierului formează o pereche, să se afișeze la ecran toate perechile de numere quasi-amiabile din fișier.

Exemplu:


numere.txt:
25 18
48 75
195 140
260 310
se afișează:
48 75
195 140


Rezolvare:


#include < bits/stdc++.h >

using namespace std;
ifstream fin ("numere.txt");

int x1,x2,i,j,s1,s2;
int main()
{

while(fin>>x1>>x2)
{

for( i = 1; i < x1; i++ )

if(x1%i==0)
  s1=s1+i;

for( j = 1; j < x2; j++ )

if(x2%j==0)
  s2=s2+j;

if(s1==x2+1 && s2==x1+1)

cout << x1 << ' ' << x2 << endl;

s1=0;s2=0;


}
return 0;

}
