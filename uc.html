Prelucrarea cifrelor unui număr


Atunci când nu cunoaștem ordinul unui număr,apelăm la un algoritm elementar ce prelucrează toate cifrele unui număr și suntem atenți dacă avem un caz special pentru numărul 0.



fin >> n;
if ( n == 0 )
prelucrare caz special 0;
 else

while( n != 0 )
{

uc = n % 10;
prelucrare uc;
 n = n / 10;

}

fout << rezultat;


Exemple algoritmi:

     a) calcularea numărului de cifre:


nrcifre = 0;
if ( n == 0 ) nrcifre = 1;
while( n != 0 )
{

nrcifre++;
n = n / 10;

}


     b) determinarea primei cifre:


if ( n == 0 ) primacif = 0;
while( n > 9 )

n = n / 10;

primacif = n;

     c) determinarea oglinditului unui număr:


ogl = 0;
while ( n > 0 )
{

cif = n % 10;
ogl = ogl * 10 + cif;
n = n / 10;

}

     d) numărul aparițiilor cifrei K:


nrap = 0;
if( n == 0 && K == 0) nrap=1;
while ( n > 0 )
{

cif = n % 10;
if ( cif == K ) nrap++;
n = n / 10;

}

Problemă:

      Se citesc perechi de numere naturale până la citirea a două valori nule. Să se determine câte dintre perechi încep cu aceeași cifră.

Rezolvare:


#include < bits/stdc++.h >

using namespace std;

int x, y, ucx, ucy, k;
int main()
{

cin >> x >> y;
while ( x != 0 && y != 0)
{

while ( x != 0 )
{

ucx = x % 10;
x = x / 10;

}
while ( y != 0 )
{

ucy = y % 10;
y = y / 10;

}

if ( ucx == ucy )

k++;

cin >> x >> y;

}

cout << k;
return 0;
}
