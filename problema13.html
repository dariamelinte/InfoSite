Problema 13:

       Din fişierul text tablou.in se citesc: un număr natural n reprezentând numărul de componente ale unui vector cu numere naturale nenule cu cel mult 9 cifre fiecare şi cele n componente ale vectorului (n≤100).
       Afişaţi elementele vectorului pe primul rând al fişierului text tablou.out, ordonate crescător.
       Determinați cel mai mare și cel mai mic element din cele n citite care au ultima cifră pară. Acestea se vor afișa, separate printr-un spațiu, pe a doua linie a fișierului tablou.out. Dacă nu există niciun element care să aibă ultima cifră pară, se va afișa în fișier mesajul Nu există!.

Exemplu:


tablou.in:
6
1431 142 1422 9009 32 72
tablou.out:
32 72 142 1422 1431 9009
1422 32


Rezolvare:


#include < bits/stdc++.h >

using namespace std;
ifstream fin ("tablou.in");
ofstream fout ("tablou.out");

int n,i,v[101],ok,j;
int main()
{

fin>>n;
for(i=1; i<=n; i++)

fin>>v[i];


for(i=1; i < n; i++)

for(j=i+1; j<=n; j++)

if(v[i]>v[j])
  swap(v[i], v[j]);

for(i=1; i<=n; i++)

fout << v[i] <<' ';

fout << endl;

for(i=n; i>=1; i--)

if(v[i]%2==0)
{

fout << v[i] <<' ';
ok=1;
break;

}


for(i=1; i<=n; i++)

if(v[i]%2==0)
{

fout << v[i] <<' ';
ok=1;
break;

}


if(ok!=1)

fout<<"Nu există!";


return 0;

}
