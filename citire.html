Citirea unui vector și afișarea sa


Vectorii sunt folosiți pentru a memora mai multe valori în același timp.
Fiecare element se accesează după poziția pe care o au în șir.

Citirea unui vector


cin >> n;
for ( i = 0; i < n; i++ )

cin >> v[ i ];



Afișarea unui vector

  • pe linie


    • for( i = 0; i < n; i++ )

    cout << v[ i ] << ' ';


  • de la ultimul la primul element


    • for( i = n - 1; i >= 0; i-- )

    cout << v[ i ]<< ' ';
