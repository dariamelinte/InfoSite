Ordonarea unui vector


I. Bubble - Sort ( folosește perechile alăturate )


do{ ok=1;

for( i = 0; i < n - 1; i++)

if( v[ i ] > v[ i + 1 ] )
{

ok = 0;
swap( v[ i ], v[ i + 1 ])

}

} while( ok == 0 );


II. Selecția directă ( folosește toate perechile posibile )


for( i = 0; i < n - 1; i++ )

for( j = i + 1; j < n; j++ )

if( v[ i ] > v[ j ] )

swap( v[ i ], v[ j ] );

Problemă:

      Se dă un vector cu n elemente numere întregi, n fiind număr par. Să se ordoneze crescător elementele din prima jumătate a vectorului și descrescător elementele din a doua jumătate.

Rezolvare:


#include < bits/stdc++.h >
