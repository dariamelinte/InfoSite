Determinarea minimului / maximului într-un vector



Pentru a afla valoarea maximă dinntr-un vector, se inițializează maximul "max" cu o valoare cât mai mică și se compară restul valorilor cu maximul. Dacă găsesc o valoare mai mare ca maximul, acesta ia valoarea respectivă. Analog este aflat și valoarea minimă.



int maxim = -999999, minim = 999999;

for( i = 0; i < n; i++ )
{

if ( v[i] > maxim )

maxim = v[i];

if ( v[i] > minim )

minim = v[i];

}


Problemă:

      Să se ordoneze un vector doar pentru elementele aflate între minimul și maximul valorilor din vectorul respectiv.

Rezolvare:


#include < bits/stdc++.h >

using namespace std;
ifstream fin ("date.in");
ofstream fout ("date.out");

int v[101],n,i,j,maxim,minim,pozitie_max,pozitie_min;

{

fin >> n;

fin>>v[i];

{ maxim=v[i];

pozitie_max=i;}

if(v[i]< minim)
{ minim=v[i];

pozitie_min=i;}

}
if(pozitie_min>pozitie_max)

swap(pozitie_min,pozitie_max);


for(i=pozitie_min; i< pozitie_max; i++)

for(j=i+1; j<=pozitie_max; j++)

if(v[i]>v[j])
  swap(v[i],v[j]);

for(i=1; i<=n; i++)

fout<< v[i] <<' ';

}
