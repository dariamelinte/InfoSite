Citirea unei matrice și afișarea sa


O matrice este folosită pentru a memora mai multe valori în același timp pe diferite rânduri și coloane.
Fiecare element se accesează după poziția pe care o au în matrice.

Citirea unei matrice


cin >> n >> m;
for( i = 1; i <= n; i++ )

for( j = 1; j <= m; j++ )

cin >> a[ i ][ j ];



Afișarea unei matrice

I. În linie


for(i=1; i<=n; i++)

for(j=1; j<=m; j++)

cout << a[i][j];


II. Fiecare linie în parte cu toate elementele de pe coloană


for(i=1; i<=n; i++)
{

for(j=1; j<=m; j++)

cout << a[i][j];
cout << endl;

}
