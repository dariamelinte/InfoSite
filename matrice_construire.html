Construirea unei matrici


I. Pe baza unei formule

Exemplu:

      a) Fiecare element este egal cu suma dintre i și j

Rezolvare:


cin>>n>>m;
for(i=1; i<=n; i++)

for(j=1; j<=m; j++)

cin>>a[i][j];


for(i=1; i<=n; i++)
{

for(j=1; j<=m; j++)

cout << a[i][j] << ' ';
cout << endl;

}

      b) Diagonalele principale și cea secundare sunt 0. Restul elementelor sunt egale cu cel mai mare indice dintre linie și coloană

Rezolvare:


for(i=1; i<=n; i++)

for(j=1; j<=m; j++)
{

if( i > j )

a[i][j]=1;

else if( i < j )

a[i][j]=0;

else a[i][j]=0;

if(i+j=n+1)

a[i][j]=0;

}


II. Când valoarea din matrice nu are legătură cu linia sau coloana.


Exemplu :



nr=1;
for(i=1; i<=n; i++)

for(j=1; j<=m; j++)
{

a[i][j]=nr;
nr++;
