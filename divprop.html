Determinarea divizorilor proprii ai unui număr natural


Un număr natural are divizori:
  • improrii: 1 și el însuși;
  • proprii: de la 2 la n / 2;
Pentru a determina divizorii proprii ai unui număr se iau toate numerele din intervalul [ 2; n/2 ] și se verifică dacă este divizorul acestuia.

Afișarea divizorilor


cin >> n;
cout << 1 << ' ';
for ( d = 2; d <= n / 2; d++)

if( n % d == 0 )

cout << d;

cout << n;

Problemă:

      Să se afișeze câți divizori proprii are un număr natural n și afișați suma acestora.

Rezolvare:


#include < bits/stdc++.h >

using namespace std;
ifstream fin ("date.in");
ofstream fout ("date.out");

int n, d, k, s;
int main()
{

fin >> n;
for( d = 2; d <= n / 2; d++)

if( n % d == 0 )
{

s = s + d;
k++;

}

fout << k << ' ' << s;
return 0;

}
